⚙️ Configurar Supabase

Para que las notas se sincronicen en tiempo real entre vos y Carlos, necesitás ingresar los datos de tu proyecto Supabase.

Tabla SQL a crear en Supabase:

CREATE TABLE notas (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  texto text NOT NULL,
  prioridad text NOT NULL,
  autor text NOT NULL,
  created_at timestamptz DEFAULT now()
);

ALTER TABLE notas ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public access" ON notas FOR ALL USING (true) WITH CHECK (true);
